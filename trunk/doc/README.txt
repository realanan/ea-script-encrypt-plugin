========== HOW TO USE ================
1. Copy easep.dll file into eAthena's plugins directory.
2. Edit conf/plugin_athena.conf file, add this line
   plugin: easep
3. Execute easetool.exe. There is a context popup menu in the window of this program, throught it, you can set password, switch the encrypt mode and decrypt mode. Drag the files you want to encrypt/decrypt into the window to encrypt/decrypt them.
4. Run the eAthena emulator, the program want you to enter the password to decrypt, and load scripts normally.

========== RUN ENVIRONMENT ===========
This plugin can run in almost all of the Windows versions of eAthena and it's inherited, except some version not support plugin.
This program also can encrypt database files in /db/ directory.
